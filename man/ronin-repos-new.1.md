# ronin-repos-new 1 "2023-02-01" Ronin Repos "User Manuals"

## SYNOPSIS

`ronin-repos new` [*options*] *PATH*

## DESCRIPTION

Generates a new git repository.

## ARGUMENTS

*PATH*
  The path to the new git repository directory.

## OPTIONS

`-h`, `--help`
  Prints help information.
